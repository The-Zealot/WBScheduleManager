Изменения в версии 3.2.4

- Реализована возможность экспортировать данные в Excel таблицу.

- Сброшен номер последней сборки.

Изменения в версии 3.0.3a

- Изменен дизайн вкладки График.

- Раздел с настройками вынесен в отдельную вкладку.

- Добавлена вкладка для описания последних обновлений Что нового?

- Изменен принцип работы перерасчета графика. Теперь, прежде чем график будет пересчитан с новыми параметрами, пользователю необходимо выбрать начальную дату отсчета.

- Изменен принцип работы оплаты всех смен. Теперь необходимо выбрать дату, до которой будут оплачены смены.

- Добавлено предупреждение во время удаления сотрудника.

- Добавлена возможность редактировать доступное расписание в разделе Настройки.

- Исправлена ошибка, из-за которой поле с информацией по смене могло быть пустым и не обновлялось при смене пункта.
